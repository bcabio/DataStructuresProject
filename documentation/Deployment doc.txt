Clone the repo from:
https://github.com/bcabio/DataStructuresProject

Download node from:
https://nodejs.org/en/download/

in the cloned repo, run `npm install` and `npm install -g typescript`,
then run `tsc` to compile typescript

Make a Google Cloud instance and import csv file of data into the SQL server. This step may be replaced with any other MySQL instance, but redirects would be required.

cd into `server` folder, and run node.js

make a request to `localhost:5000` in a browswer\
